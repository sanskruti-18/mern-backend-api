# MERN Backend API

A backend REST API built using **Node.js**, **Express.js**, and **MongoDB** for managing posts with full CRUD operations.

## 📌 Features

- Create a post
- Get all posts
- Update a post
- Delete a post
- MongoDB database integration
- Middleware validation
- Proper HTTP status codes
- Clean folder structure (MVC + Service + Repository)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman
- JavaScript
